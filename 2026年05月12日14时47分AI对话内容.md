## 关键技术总结

### AtomCode API 架构

```
用户 → 本代理 (localhost:8000) → api-ai.gitcode.com/v1 (OpenAI兼容格式)
                                  ↑ User-Agent: atomcode/4.22.0 (独占模型必需)
                                  ↑ Authorization: Bearer {access_token}

模型列表: api.gitcode.com/api/v5/coding-plan/models
额度状态: api.gitcode.com/api/v5/coding-plan/status
Token刷新: acs.atomgit.com/oauth/refresh
```

### 认证流程

1. `atomcode login` → 浏览器 OAuth → 写入 `~/.atomcode/auth.toml`（access_token + refresh_token）
2. 代理读取 auth.toml 中的 access_token
3. Token 过期时自动通过 refresh_token 刷新
4. 请求上游时添加 `Authorization: Bearer {token}` + `User-Agent: atomcode/4.22.0`
